## Getting Started

1. **Get API Key**: [onchain.fi/get-api-key](https://onchain.fi/get-api-key)
2. **Install**: `npm install @onchainfi/uplink`
3. **Set Environment Variables**:
   ```bash
   export ONCHAIN_API_KEY=your-key
   export UPLINK_PRIVATE_KEY=0x...
   ```
4. **Make a payment**: See [JavaScript README](./javascript/README.md)
